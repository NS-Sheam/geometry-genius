What is DOM? What is the purposes of DOM?

DOM full form is Document Object Model.It a programming interface for web documents.The DOM represents the document as nodes and objects. Javascript can interact web page with DOM .

How will you select HTML elements using DOM?

We can select HTML elements using DOM by these methods- getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(),querySelectorAll().

What is event bubble?

Event bubble is a method of event propagation in the HTML DOM when an event is in an element inside another element, and both elements have registered a handle to that event.

What is callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is called by that function when it completes its task. We use it to avoid writing same code repetedly.
